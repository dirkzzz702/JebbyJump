## What to redesign

**Top HUD (a thin bar across the top):**

- **Level badge** (top-centre): a dark pill "Level 10" today → a small cream /

- **Timer** (top-right): e.g. "00:07.34" → set in a small cloud or ribbon plate.
- **Pause button** (top-right): a plain circle today → a round cream/gold button

**Centre:**
- **Hint / toast message** (short cues like "Wait for Go!", memory prompts): a
  soft cloud or ribbon banner, warm and very readable.

**Result panels (dark cards today):**
- **Level Complete card** — title "Level Complete!", then Time, Best, **Rank
  (S / A / B / C)**, **Stars (x/3)**, and three buttons (Retry / Next Level /
  Main Menu). Reskin to a **light cream cloud-kingdom card** matching the menu
  (warm-brown outline, cloud/vine corner accents), with buttons in the menu's
  plate style. Rank + stars should feel celebratory and read clearly on the light
  card.
- **Game Over card** — title "Game Over", two buttons (Retry / Main Menu). Same
  card style; keep it gentle and encouraging, never harsh (this is a kids' game).
